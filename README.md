# Swasthya Mitra

Swasthya Mitra is a health-tech web application designed to assist users with disease prediction, precautionary measures, and medical guidance based on symptoms. This project is developed collaboratively and follows a modular architecture.

1. **User Authentication System**
   - Signup and login functionality for users.
   - Secure authentication using Spring Security and JWT (planned).

2. **Disease Prediction from Symptoms**
   - Input symptoms using speech-to-text (voice-enabled input).
   - Use of machine learning models to predict possible diseases based on symptoms.

3. **Precaution and Remedies**
   - Display a list of precautions and quick remedies based on predicted disease.
   - Provides educational content and health guidance.
